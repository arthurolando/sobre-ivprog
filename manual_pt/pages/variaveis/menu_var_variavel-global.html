Variável Global

As variáveis servem para armazenar valores. No caso de variável global, é toda variável criada na área externa do programa principal (vide imagem). Esse tipo de variável é reconhecida em qualquer ponto de seu código, ou seja, dentro de um função qualquer, pode-se usar essa variável desde que não não exista uma variável local com o mesmo nome! Recomenda-se que evite usar esse tipo de variável, pois elas podem gerar confusão no código.

Ao criar uma variável global, pode-se configurar as seguintes características:

  1. Const: embora tenha o nome de variável, quando definimos uma variável como sendo constante, ela não poderá ter seu valor inicial alterado;
  2. Tipo: define o tipo de informação que será armazenada na variável (tipos simples: inteiro, real, cadeia ou lógico; vetor: de qualquer tipo simples)
  3. Nome: define o nome da variável, utilizado para acioná-la de qualquer parte do programa;
  4. valor: define um valor para ser armazenado na variável.
