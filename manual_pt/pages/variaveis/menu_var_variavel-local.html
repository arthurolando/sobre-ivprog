Variável Local

As variáveis servem para armazenar valores. No caso de variável local, é toda variável criada dentro de determinada função. Nesse caso, ela será conhecida apenas dentro dessa função. Recomenda-se que utilize apenas esse tipo de variável, evitando-se as globais, pois essas última podem gerar confusão no código.

  1. Tipo: define o tipo de informação que será armazenada (inteiro, real, cadeia, lógico e vetor)
  2. Nome: define o nome da variável, utilizado para acioná-la de qualquer parte do programa
  3. valor: define um valor para ser armazenado na variável
