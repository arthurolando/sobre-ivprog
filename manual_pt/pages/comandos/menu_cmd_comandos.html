Comandos

Os comandos são utilizados para criar a estrutura lógica de um programa. Na interface visual do iVProg, os comandos podem ser inseridos no código a partir do botão , basta passar o "mouse" sobre esse botão, aparecerá a lista dos comandos que podem ser utilizados, como ilustrado na figura 1. são No iVProg você poderá adicionar os seguintes comandos:

  • Comandos principais
    • entrada de dados (leitura);
    • saídas de dados (impressão);
    • inserção de comentário;
    • atribuição (definir valor para alguma variável);
    • chamar função (usada para gerar um valor);
    • comando de seleção;
    • comando de repetição fixo (laço do tipo para);
    • comando de repetição com condição de entrada (laço do tipo repita enquanto);
    • comando de repetição com condição de saída (laço do tipo repita até que);
    • comando de seleção dentre várias opções;
    • comando de devolução (usado em funções).

Fig. 1. Principais comandos do iVProg, destando a mensagem que aparece ao deixar o "mouse" sobre a primeira opção.
  • Comandos contextuais
    Os comandos de repetição admitem um comando especial para "quebrar" a repetição, quer dizer, se ele for executado o laço é interrondido (Pare).
    Outro comando especial é o caso, que deve ser usado dentro de um comando de escolha, como na figura 2.

    • caso
