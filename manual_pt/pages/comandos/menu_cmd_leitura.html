Leitura de Dados

Esse comando é utilizado para receber uma entrada a ser digitada pelo usuário. Pode ser um número (inteiro ou flutuante) ou cadeia de caracteres.

Na figura 1 apresentamos um exemplo de código com um comando para leitura ou entrada de dados. Nesse programa existe apenas uma variável, o que é necessário para armazenar o valor a ser digitado pelo usuário durante a execução do código. No exemplo usamos também um comando para saída de dados ou impressão.
Na imagem também aparece a área em que existe a troca de informações entre o programa e o usuário, o terminal. Para a imagem gerada, o programa foi executado duas vezes, na primeira o usuário digitou o valor 4 (daí a impressão O valor digitado foi:4) e na segunda vez, foi digitado o valor -128.
